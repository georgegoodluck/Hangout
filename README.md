# 📍 Hangout

Welcome to **Hangout**, a web application designed to help people connect and organize social gatherings. This repository contains the source code and documentation for the project.

## 📂 Project Structure

This repository is organized to keep the core application and its identifiers separated:

* **`/hangout`**: The main application source code (React/Next.js).
* **`/hangout-ids`**: Component identifiers and specific project assets.
* **`LICENSE`**: Legal information regarding the use of this code.
